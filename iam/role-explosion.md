# 🚨 Problem: Role Explosion / Confusing Roles

## Description
Too many roles with overlapping or unclear permissions.

---

## ⚠️ Risk
- Hard to track permissions → misconfigurations  
- Users may get excessive privileges  

---

## 🔍 Scenario
An organization had 50+ roles for 20 users.  
One user was assigned multiple roles, accidentally gaining admin-level access.

---

## ✅ Fix / Best Practice
- Maintain a minimal set of well-documented roles  
- Use role naming conventions  
- Periodically audit role assignments  

---

## 🧠 Lesson Learned
Simplicity + clarity in roles prevents accidental over-permission.
