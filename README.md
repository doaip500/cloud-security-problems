# cloud-security-problems
A collection of real-world cloud security misconfigurations, risks, and practical fixes — documenting my journey in IAM, AWS, and DevSecOps.

This repository contains practical cloud security problem scenarios to help engineers:
Understand common misconfigurations in IAM, S3, EC2, and Docker
Learn how to identify, assess, and fix vulnerabilities
Apply least privilege, secure configurations, and cloud best practices
Track my learning journey in cloud security, DevSecOps, and identity management
