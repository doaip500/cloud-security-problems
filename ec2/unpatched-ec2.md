# 🚨 Problem: Unpatched EC2 Instance

## Description
EC2 instances running outdated OS or software with known vulnerabilities.

---

## ⚠️ Risk
- Exploitable vulnerabilities → remote compromise  
- Malware / ransomware risk  

---

## 🔍 Scenario
A server running outdated packages was scanned and exploited by a botnet.  
Critical data was stolen.

---

## ✅ Fix / Best Practice
- Enable automated updates  
- Regularly patch OS and software  
- Monitor vulnerability reports  

---

## 🧠 Lesson Learned
Patch management is fundamental to cloud security.
